MIRACL: Metadata Indexing and Retrieval for Automated Context Learning
Overview

MIRACL (Metadata Indexing and Retrieval for Automated Context Learning) is an AI-powered document intelligence system designed to transform unstructured documents into a searchable knowledge base. The system extracts text from documents, generates semantic embeddings, stores them in a vector database, and enables natural language querying using Retrieval-Augmented Generation (RAG).

This project demonstrates how modern AI frameworks such as LangChain, LlamaIndex, and vector databases can be used to build scalable information retrieval systems capable of answering questions based on document content.

Key Features:

Document Ingestion – Load and process PDF documents.

Text Chunking – Break large documents into manageable segments for efficient retrieval.

Vector Embeddings – Convert document chunks into semantic embeddings using transformer models.

Vector Database – Store embeddings in ChromaDB for fast similarity search.

Retrieval-Augmented Generation (RAG) – Retrieve relevant document chunks and generate answers using a language model.

Interactive Query System – Users can ask questions about uploaded documents and receive contextual answers.
